# PipeWire 7.1 Virtualization

[Bazzite](https://bazzite.gg) ships a convenience command that handles this setup:

```bash
ujust setup-virtual-surround
```

The SOFA file can be swapped as needed. See the [SOFA file repository](https://www.sofaconventions.org/mediawiki/index.php/Files) for alternatives.
