# Proton Configuration

A list of games I've tested on Bazzite.

| Game | Launch Options | Notes |
|---|---|---|
| Granblue Fantasy: Relink | *out of the box* | |
| Cyberpunk 2077 | `--launcher-skip` | technically runs out of the box |
| God of War: Ragnarok | `SteamDeck=0 WINEDLLOVERRIDES="version=n,b" %command%` | Requires `PsPcSdk.dll` and `version.dll` from NoPSSDK |
| The Coffin of Andy and Leyley | *out of the box* | |
| Wuthering Waves | `-SteamOS=1 -SteamDeck=0 %command%` | `SteamOS=1` enables anti-cheat; disabling SteamDeck exposes full graphics settings |
| Blue Archive | *out of the box* | |
| Clair Obscur: Expedition 33 | *out of the box* | previously requires `sd0` to disable Steam Deck graphics profile |
| Resident Evil 9 | *out of the box* | |
| DOOM: The Dark Ages | *out of the box* | |
